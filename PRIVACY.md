# Kioku — Privacy Policy

**Last updated: July 18, 2026**

Kioku's entire purpose is to keep your data **yours**. It does not collect,
transmit, sell, or share any data. Everything stays on your device.

- **What Kioku stores — locally only:** copies of your own AI chat
  conversations (from claude.ai, chatgpt.com, gemini.google.com), your
  folders, and the memory blocks you write. All of it lives in your browser's
  local extension database (IndexedDB) on your computer. It is never uploaded
  anywhere. Deleting the extension deletes the archive.
- **What leaves your device: nothing.** No conversation content, no titles,
  no metadata, no identifiers, no analytics, no crash reports. There is no
  Kioku server.
- **Settings sync:** your preferences (e.g. performance-mode on/off) use
  Chrome's built-in `storage.sync`, controlled by you and Google — not by us.
- **Network requests:** the extension periodically downloads one small public
  JSON file (styling/selector fixes) from a static GitHub URL, so archiving
  keeps working when the AI sites change their design. This request contains
  no user data and no cookies, and the file contains only configuration data —
  never code.
- **Permissions:** Kioku runs only on claude.ai, chatgpt.com/chat.openai.com,
  and gemini.google.com, and only to read the conversation you are viewing so
  it can be archived for you.
